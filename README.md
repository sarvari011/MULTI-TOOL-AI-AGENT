# 🤖 Multi Tool AI Agent

A simple AI Agent built using Python, Ollama, Qwen 2.5, and Streamlit.

## Features

* Wikipedia Search Tool
* Weather Information Tool
* Calculator Tool
* Streamlit Chat Interface
* Agent Workflow:

  * Plan
  * Action
  * Observation
  * Output

## Technologies Used

* Python
* Ollama
* Qwen 2.5
* Streamlit
* Wikipedia API
* Requests

## Project Structure

├── agent.py
├── app.py
├── README.md
└── .gitignore

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Start Ollama

ollama run qwen2.5:3b

3. Run Streamlit App

streamlit run app.py
